Markdown 学习

  • Markdown的安装就直接用了Vscode自带的Markdown编辑器，顺便下了一个实时预览的插件Markdown Preview,使用起来也是蛮方便的
  • 至于Markdown语法的学习，比较推荐菜鸟教程的Markdown教程
  • Vscode这个Markdown插件可以改变实时预览窗口的css样式，还可以导出为pdf等格式，真的非常好用，强力推荐ヾ(❀╹◡╹)ﾉ~

    Linux 学习

  • Linux 安装
    • 虚拟机的安装
    • Linux系统的安装
    • 比较推荐csdn上的这篇文章Linux安装教程（超详细版)，下载的是CentOS7，如果想要安装别的版本可以另见别的教程
    • 任务中的输出需要neofetch这个命令，需要通过包的安装来完成这个内容（注意在安装时，需要开启root权限，也就是sudo su这个指令）

      Git 学习

  • Git的下载安装都比较简单，就不做赘述了，Git的使用也是很方便，是一个命令行的格式，然后针对任务需要的命令就直接善用搜索引擎去搜索然后输入进去就可以了，在搭建博客的时候也有用到
  • 具体的学习可以参考Git教程(也就是工作室网页上那个XD)

    Vscode 配置

  • 详情参考挑把趁手的兵器——VSCode配置C/C++学习环境（小白向）知乎专栏(也是工作室网页同款)
  • 这个配置文件中有几个步骤都可能出错，可能是我比较幸运没有遇到这样的问题ovo

    Blog 搭建

  • 博客的搭建强推csdn上的一篇文章,讲的内容很全，而且很多步骤都有截图，可以自查自己那里出了问题hexo史上最全搭建教程
  • 在上述教程的最后一步，也就是上传文章的那步，我当时就因为报错卡了好久，可以尝试修改xml文件deploy中repo链接，然后就可以了，也算是这个教程中有一点疏漏的地方吧，不过每个人在搭建的时候会出现不同的问题，所以善用搜索引擎还有报错的提示来解决问题吧
